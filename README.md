# 🎓 International Student Journey Analytics

## 📌 Project Overview

This project analyzes the journey of international students from visa completion to retention using a synthetic dataset.

The goal is to identify drop-off points, measure engagement, and provide actionable insights to improve student success.

---

## 🧠 Problem Statement

Universities often struggle to track and optimize the international student journey.

Key challenges include:
- Students not attending orientation
- Low engagement in campus activities
- Drop-offs before retention

This project simulates that journey and analyzes it using KPIs and SQL.

---

## 📊 Dataset

The dataset represents student lifecycle stages:

- Visa Completed
- Arrival Completed
- Orientation Attended
- Check-in Completed
- Event Participation
- Retention

Location: `data/student_journey_data.csv`

---

## 📈 Key KPIs

- Orientation Attendance Rate
- Engagement Rate
- Retention Rate
- Document Completion Rate
- Funnel Conversion Metrics

Detailed definitions: `docs/kpi_definitions.md`

---

## 💻 SQL Analysis

All KPI queries are written in:

`sql/kpi_queries.sql`

- Rate calculations

---

## 🔍 Key Insights

- Orientation attendance is a major driver of retention
- Student engagement strongly impacts outcomes
- Early-stage drop-offs exist between visa and arrival
- Operational steps like check-in affect long-term success

Detailed insights: `insights/journey_insights.md`
